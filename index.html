Config Options :


No Config nothing, as such

Usage :


HTML simple input tags, with data-check="parent" attr for parent, data-check="child" attr for children, and data-name="name" attr for grouped checkboxes.

<div class="group">
    <input type="checkbox" data-check="parent" data-name="name">
    <div class="children">
        <input type="checkbox" data-check="child" data-name="name">
        ...
    </div>
</div>

JS simple plugin call, no options needed

$('input[data-check="parent"]').multicheck();
